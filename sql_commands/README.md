# SQL Commands Folder

Este diretório foi criado para você guardar os comandos SQL que testar nas suas aulas.

Como usar:

- Crie arquivos com extensão `.sql` ou `.txt`.
- Salve consultas que você quiser revisar depois.
- Use o notebook para executar as consultas e, se quiser, copie o conteúdo para os arquivos de comando.

Exemplo de arquivo:

```sql
SELECT Nome, Sexo, Total
FROM baby_names
ORDER BY Total DESC
LIMIT 10;
```

Dica: mantenha seus `comandos.sql` organizados por aula ou por tema (select, join, window functions, etc.).
