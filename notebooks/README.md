# Notebooks

Este diretório pode ser usado para guardar notebooks de aula e exemplos adicionais.

Como usar:

- Salve cada aula ou exercício como um notebook separado.
- Use nomes claros como `aula_window_functions.ipynb` ou `aula_joins.ipynb`.
- Mantenha o notebook principal `sql_window_functions_ranking.ipynb` como o exemplo inicial de uso.
